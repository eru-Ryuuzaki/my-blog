# 关于看网课好和看书的选择

- 我更加喜欢**先看网课**，知道大概怎么操作，有个编程思路
- 然后就是**边看边敲**，跟着做一些小demo，有助于加深记忆和更加容易发现自身问题
- 在了解了基本使用之后，**回归课本或者深入看官方文档**，有利于更加全面地去掌握这门技术
## 看网课 or 看博客
- 优点：学起来不那么累，入门快
- 缺点：可能存在不严谨的地方
## 看书
- 优点：知识比较全面，容易形成完整的知识体系，有权威性
- 缺点：有些内容比较枯燥，平时实战用的也少，所以忘得也快
